# NLU Doctoral Symposium — May 2026

This repository is intended to host the standalone HTML presentation:

**Beyond Secondary Orality — NLU Doctoral Symposium**

## Publishing with GitHub Pages

After `index.html` is added to the repository root:

1. Go to **Settings → Pages**.
2. Under **Build and deployment**, select **Deploy from a branch**.
3. Select branch: `main`.
4. Select folder: `/root`.
5. Save.

The site should publish at:

`https://minerclass.github.io/nlu-docsymposium-may26/`
